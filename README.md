## Local Development

1. Clone the repository:
```bash
git clone https://github.com/your-username/SEG3125_A5_Connor.git
cd SEG3125_A5_Connor
```

2. Install dependencies:
```bash

```

3. Start development server:
```bash
npm run dev
```

4. Build for production:
```bash
npm run build
```
